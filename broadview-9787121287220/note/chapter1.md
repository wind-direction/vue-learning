# 第一章 遇见Vue.js

## 1.2 Vue.js 是什么？

### Vue的特性

- 确实轻量
- 数据绑定
- 指令
- 插件化

### 与Angular的区别

|            |AngularJs |           VueJs         |         reactJs           |
|:-----------|:---------|:------------------------|:--------------------------|
|指令        |          |                         |                           |
|过滤器      |          |                         |                           |
|双向邦定    |          |                         |                           |
|低端浏览器  |          |                         |                           |
|学习成本    |高        |低                       |                           |
|更新技术    |脏检查    |依赖追踪,异步队列更新    |V-DOM 脏检查               |
|编译前文件  |          |.vue                     |.jsx                       |
|钩子        |          |支持                     |支持                       |
|插件        |          |                         |                           |
|V-DOM       |          |原生DOM                  |支持V-DOM,会进行脏检查     |
